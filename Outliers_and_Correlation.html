4. Understanding Correlation

Correlation tells how different columns in the dataset are related to each other.

To define Correlation: One unit change in one coulmn will translate to how many units change in another column.

There are 3 types of Correlation:

Positive correlation: Direct proportionality b/w 2 columns. 
                      If one increases another increases, if one decreases another decreases.

Negative correlation: Inverse proportionality b/w 2 columns. 
                       If one increases other decreases, if one decreases other increases.

No correlation      : No relation or dependancy b/w 2 columns. 
                      One's increase or decrease does not affect the other column.

4.2 Heatmap

One alternate EDA which is more frequently used to check correlation is called as 'heatmap'.

Heatmap is a matrix plot of correlations between columns within the data.

In a heatmap, The value of Correlation spans b/w -1 0 and 1:

-1 or closer values: Negative correlation

0 or closer values: No correlation

1 or closer values: Positive correlation

We use seaborn as a library to visualize heatmap. The syntax used is : sns.heatmap(dataset_name.corr(), annot= True).

.corr() is a method used to plot the correlation matrix, whereas annot = True is used to annote values (dispay the correlation values) on the matrix.

In [40]:
import seaborn as sns
sns.heatmap(data.corr(), annot=True) # annot means annotation, it will assign values to the heatmap.
Out[40]:
<Axes: >

Let us interpret the heatmap above.

The strip on it's right side gives us the value of correlation. Lighter shade (values closer to 1.0) gives higher correlation and Darker shade (values closer to 0.2) gives lower correlation.

The data in the diagonal will always have correlation 1 because it shows the correlation of the column with itself. For e.g Sales (X-axis) = Sales (Y- axis), Newspaper (X-axis) = Newspaper (Y- axis)... and so on....

If we check the correlation of different columns wrt Sales we get,

0.16 for Newspaper

0.35 for Radio

0.9 for TV (close to 1 hence a good positive correlation)

From the data above one can conclude that if one invests in marketing on 'TV' products the Sales will increase for sure.
